Como hostear gratis

1. GitHub Pages (recomendado):
Sube este archivo HTML a un repositorio en GitHub, ve a Settings > Pages, selecciona la rama main y el archivo index.html. Tu pagina estara disponible en tuusuario.github.io/nombre-repo en minutos.


2. Netlify Drop:
Abre netlify.com/drop, arrastra la carpeta con tu archivo HTML y listo. Te da una URL publica al instante. Sin registro obligatorio.


3. Vercel:
Conecta tu repo de GitHub con vercel.com y despliega automaticamente con cada cambio que hagas.


Nota: todos los datos se guardan en el navegador (localStorage). Para compartir datos entre miembros en tiempo real, considera agregar Firebase Firestore (gratuito) mas adelante.
